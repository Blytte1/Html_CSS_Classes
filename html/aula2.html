Formatação

Para podermos fazer negrito usamos entre tags "Strong", assim o trecho ficará mais escuro. ou entre tags "b."

Usamos a tag "i" para Itálico e tag "u" para Underline

O "del" serve para fazer esse risquinho no meio.

O "mark" destaca textos, embora isso seja melhor fazer por css.

A tag "small" cria pequenos textos.

A tag "sup" cria textos como se fossem expoentes.

A tag "sub" cria textos inferiores como esse.

Para fazer a linha abaixo usamos a tag "hr."
